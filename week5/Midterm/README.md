# wwcc-cpp
The name of this project is "Movie Database Collection."
In this project i've included 6 different options depending on the number the user enters from 1-6, with each number performing certain tasks.
When entering 1. The user must enter certain movie characteristics including title, director, genre, ratings, and year. Once entered, the movie is then stored in to the system to keep track of including any other movies the user decides to input.
When entering 2. The user can search for a movie by entering the genre of the movie at which they wish to look for.
When entering 3. The user can filter through the movies that they have prevouisly enter using a range of years, an example would be from 2016 - 2026.
When entering 4. The user can see the total number of movies that they have entered into the systen as well as displaying the average ratings of all the movies entered.
When entering 5. The user can actively see all movies they have entered and have the option mark a movie as 'seen' by entering the exact title of the movie. Once entered, the system will then remove the movie automatically from the watchlists.
When entering 6. The user exits out of the code.
