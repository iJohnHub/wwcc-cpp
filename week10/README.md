# Prisoner's Break - An Interactive Story Game

# How To Play
1. Run the compiled executable in your terminal.
2. Upon launching the game, you'll be prompted to name your character.
3. Each scene will present a brief narrative on which there'll be choices the user can select.
4. Input the number that matches your desired action.
5. Win/Loss Conditions: 
    **Win**: You successfully smuggle yourself out of the dungeon and have earned your freedom.
    **Loss**: You loss all your health and/or get spotted trying to escape.
